TV Schedule Widget 

- part of Multimedia Information Systems 1 course on TU Graz. (without MVC)

- TV Schedule Widget is made for displaying TV broadcast schedule for some Austrian and
German television stations. It covers TV schedule for next three days and can be refreshed
by user interaction. The widget have possibility of displaying current content shown on
selected broadcast station and a full-day content for the next three days, for selected
broadcast station. It uses Web Service for generating data (.json files) which is used for
all content displayed on the widget.
